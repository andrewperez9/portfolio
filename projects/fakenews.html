Fake News: Machine Learning Project

Fake News and misinformation is rampant.

In the modern social media age, we are inundated with misinformation that is often small and given without context. These lies have major ramifications on our lives. They are conveyed through headlines, social media posts, and comments, often small and containing as little as 10 words.

Project Description

This project aimed to develop a system that can determine the truthfulness of a statement given minimal information. We used Natural Language Processing techniques and the following Machine Learning algorithms in Python to accomplish this:

  • K-Nearest Neighbors (KNN)
  • Convolutional Neural Network (CNN)
  • Logistic Regression
  • Support Vector Machine (SVM)
  • Recurrent Neural Network (RNN)

Dataset Description

Kaggle Dataset

This dataset contains over 11,000 statements along with related key information and, most importantly, the degree of truthfulness.

  • Label (truthfulness)
    • true
    • mostly-true
    • half-true
    • barely-true
    • false
    • pants-on-fire
  • Statement
  • Speaker (name)
  • Credit History (past counts of truthfulness)

Example:


Label: true
Statement: "Putting three Republicans in my Cabinet... is unprecedented."
Title: President of the United States, Affiliation: democrat
Credit History: (70 barely-true, 71 false, 160 half-true, 163 mostly-true, 9 pants-on-fire)

Model: Accuracy%

  • K-Nearest Neighbors: 21.7%
  • Convolutional Neural Network: 22.0%
  • Logistic Regression: 24.0%
  • Support Vector Machine: 25.1%
  • Recurrent Neural Network: 27.0%

Tools Used

  • NumPy
  • TensorFlow
  • NLTK
  • Word2Vec
  • TextBlob
